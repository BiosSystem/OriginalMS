# Welcome to the OriginalMS Wiki

**OriginalMS** is a Dockerized classic MapleStory v62 (GMS 2008) server emulator built on Java 8 and MySQL.

- **[Troubleshooting](Troubleshooting)** - Common setup issues, client connection fixes, and database notes.
